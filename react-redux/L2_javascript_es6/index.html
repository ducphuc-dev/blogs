1. let & const

let và const là 2 kiểu khai báo biến mới thay thế cho var. Bạn có thể dùng let thay thế cho var và dùng const để khai báo một biến có giá trị mà giá trị sẽ không bao giờ thay đổi.

Xem thêm về let : https://developer.mozilla.org/vi/docs/Web/JavaScript/Reference/Statements/let

Xem thêm về const : https://developer.mozilla.org/vi/docs/Web/JavaScript/Reference/Statements/const

3. Exports & Imports

Trong những dự án bạn làm việc với JavaScript, mà cụ thể ở đây là ReactJS, bạn sẽ tiến hành chia nhỏ các đoạn mã của mình thành nhiều tập tin javascipt để có thể dễ quản lý, sửa đổi sau này. Chúng được gọi là modules.

LƯU Ý:
- Đối với default export: Bạn có thể import nó với bất kì tên biến mới nào bạn muốn, không nhất thiết phải trùng tên với tên biến mà bạn export
- Đối với name export: Bạn phải import nó với tên biến như tên biến mà bạn export, vì chỉ có như vậy, JavaScript mới biết bạn đang import từ biến nào trong rất nhiều biến được export
- Trong trường hợp bạn muốn sử dụng 1 tên biến mới cho biến mà bạn đã export, bạn có thể sử dụng từ khoá as
- Bạn có thể sử dụng kí tự * để import tất cả các biến, hàm,… trong 1 tập tin, và kết hợp với as để tạo thành 1 đối tượng (object) các biến, hàm,…
